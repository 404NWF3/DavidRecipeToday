# 快速开始

## 打开演示

1. 进入 `presentation` 文件夹
2. 双击 `index.html` 文件
3. 演示将在默认浏览器中打开

## 键盘快捷键

| 按键 | 功能 |
|------|------|
| `→` 或 `↓` | 下一页 |
| `←` 或 `↑` | 上一页 |
| `空格` | 下一页 |
| `Home` | 回到首页 |
| `End` | 跳到末页 |

## 特性

- ✅ 10页精心设计的幻灯片
- ✅ 完整的深色/浅色模式支持
- ✅ 响应式设计(支持手机/平板/桌面)
- ✅ 平滑的翻页动画
- ✅ 实时进度条显示
- ✅ 触摸滑动支持(移动端)
